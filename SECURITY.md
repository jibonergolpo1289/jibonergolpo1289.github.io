# Security Policy

## Supported Versions

This is a personal website/blog, not a versioned software project. 
The site currently live at the main branch is the only version maintained and updated.

## Reporting a Vulnerability

If you discover a security vulnerability or issue with this website 
(e.g. exposed data, broken authentication, injection vulnerabilities), 
please report it by:

- Opening an issue in this repository, or
- Emailing: ra1289zih@gmail.com

I will try to respond within a few days. Please avoid publicly disclosing 
the issue until it has been addressed.
